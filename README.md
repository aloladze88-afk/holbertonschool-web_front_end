# Advanced HTML

This project introduces the basics of HTML and how to structure a web page correctly. The goal is to build simple HTML pages without CSS or styling, focusing only on clean, semantic, and valid HTML.

## Learning Objectives

By the end of this project, I should be able to explain:

* how to create the skeleton of an HTML5 page;
* how to use semantic tags such as `header`, `main`, `footer`, `nav`, `section`, `article`, and `aside`;
* the difference between `div` and `span`;
* how to use headings, lists, tables, links, images, video, audio, and embedded content;
* how to structure a valid HTML page that follows W3C guidelines.
